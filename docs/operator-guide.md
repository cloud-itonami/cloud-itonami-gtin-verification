# Operator Guide

## First Deployment

1. Register with GS1 (or the relevant regional GS1 Member Organization) if
   your function requires an official prefix/allocation authority.
2. Import any existing product/brand data into your own store.
3. Run the advisor in read-only/validation-only mode.
4. Compare recommendations with current operating practice.
5. Enable human-approved writes once the GTIN Verification Governor contract is trusted.

## Minimum Production Controls

- customer/brand-owner-owned data store
- clear provenance for every identifier issued or validated
- approval workflow for any customer-facing or public-record action
- incident contact for operators
- monthly audit export

## Certification

Certified operators must prove data provenance, audit traceability, GS1 (or
equivalent) membership status where applicable, and that automated actions
cannot bypass the GTIN Verification Governor.
